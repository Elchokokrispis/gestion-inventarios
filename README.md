# gestion-inventarios
Sistema de gestión de inventarios - Proyecto integrador

## Info
Sistema integral para la gestion de inventarios, control de stock, alertas automaticas y generacion de reportes.

## Tecnologias / Herramientas
- Base de Datos: Mysql
- Backend: C# .Net Windows Forms
- Control de Versiones: GitHub

## Estructura del Proyecto
/database - Scripts SQL (Aqui iran las tablas, procedimientos, tiggers, etc.)
/src - El codigo Fuente de la aplicacion C#
/docs - Documentación y diagramas
